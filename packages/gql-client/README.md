# gql-client

A simple [Apollo Client](https://www.apollographql.com/docs/react/) that issues GraphQL requests

## Development Guide

### Getting started

```shell
npm install
```

### Test

```shell
npm test
```

### Lint check

```shell
npm run lint
```

### Lint auto-fix

```shell
npm run lint-fix
```

### Build

```shell
npm run build
```

### Clean up build and coverage artifacts

```shell
npm run clean
```
