### Getting Started

```shell
npm install
npm run watch
```

### Test

```shell
npm test
```

### Lint check

```shell
npm run lint
```

### Lint auto-fix

```shell
npm run lint-fix
```

### Build

```shell
npm run build
```

### Clean up build and coverage artifacts

```shell
npm run clean
```

### Push to origin non-master branch

```shell
npm run push
```
